# SDLC Model

## Selected SDLC Model

The Agile Software Development Life Cycle (SFLC) was selected for this project.

## Why Was Agile Selected 

Agile was selected because it allows the project to be completed one step at a time. Since the HealthBridge Telehealth Platform has many features like appointment scheduling, patient forms, secure logins and system integrations. It makes it easier to work on features before moving to the next one.

Even if this project is being completed by one person, Agile is still the best option because it helps to keep the project organized and allows for any changes to be made if needed.

## Benefits of Agile

Agile will help with:

- Break the project into smaller tasks.
- Test the system throughout the development.
- Find and fix problems earlier.
- Make improvements after each assignment.
- Keep the project organized.

## Adapting the Model

This project will follow the Agile model by completing each assignment at a time. The first assignment will focus on planning and design, then it will focus on development, testing and improving the system.  

If there are any changes needed during the project, they could be added in later assignments without having to start over. This is what makes Agile the best fit for this project.
